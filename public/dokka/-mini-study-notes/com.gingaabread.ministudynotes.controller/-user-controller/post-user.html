postUser

@PostMapping(value = [/{username}/{email}])
fun postUser(@PathVariable(value = username) username: String, @PathVariable(value = email) email: String): ResponseEntity<String>
Content copied to clipboard

Creates a new user with the specified username and email if neither the username nor email already exists.

Return

200 OK - if the username and email are unique and the new user has been therefore created; 400 BAD REQUEST - if either (or both) the username or email already exist

Parameters

username

  • the unique name of the user

email

  • the unique email of the user
